# vue_twitter clone coding
## 실행 방법
```npm run dev``` 실행 후 http://localhost:3000/ 접속
아이디: test@test.com / 비밀번호: test123 으로 로그인 후 확인 가능합니다.
또는 회원가입 후 로그인해서 테스트 하실 수 있습니다.

## 기술 스택
Vue3
Vite
Firebase
tailwind


## 페이지
- 회원가입
- 로그인
- 홈
- 알림
- 쪽지
- 프로필
- 트윗

## 페이지별 기능
### 회원가입
- 유효성 체크
- 회원가입

### 로그인
- 유효성 체크
- 로그인

### 홈
- 트윗 등록
- 다른 유저 트윗 불러오기

### 알림
- 팔로잉 한 유저의 최근 30일 트윗 불러오기

### 쪽지
- 선택한 유저의 정보 불러오기
- 유저와 채팅하기

### 프로필
- 로그인중인 유저의 정보 불러오기
- 프로필 수정 > 배경사진, 프로필 사진 변경
- 내가 쓴 트윗 불러오기
- 리트윗 불러오기
- 좋아요한 트윗 불러오기

### 트윗
- 트윗과 탑글 내용 불러오기

### 공통
- 트윗에 답글쓰기
- 리트윗 버튼 누르면 프로필 > 리트윗 탭에 추가
- 좋아요 버튼 누르면 프로필 > 마음에 들어요 탭에 추가
- 트윗 삭제하기
- 트윗 모달창으로 트윗 등록하기
- 로그아웃

### 보완점
프로필 > 리트윗, 좋아요 탭에 내림차순 정렬 필요
